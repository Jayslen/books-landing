
# Landing page - Mis libros pendiente.

En esta landing page encontraras lo que son mi lista de libros, divididos en secciones de leyendo, completados y pendientes. En esta landing podras visualizar todos estos libros y detalles del mismo con un dise√±o sutil y simple. Esta pagina fue desarrollada con astro y tailwind css y usa notion como base de datos para almacenar los datos.

## Features

- Desarrollada con astro.
- Utiliza Dynamic routes 
- ViewTransition API
- Notion como base de datos para almacenar los libros.
